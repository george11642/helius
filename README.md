# Helius

**The AI that works when nothing else does.**

Helius is an offline-first, installable web app (PWA) that runs **Gemma 4 E2B entirely on your device** — no cloud, no signal, no account. It's a voice-capable agent that chains real local tools (offline maps, routing, sun clock, signaling) to get a lost hiker back to the trailhead before dark.

Built solo, from scratch, entirely during the **RAISE Summit Hackathon 2026** (July 4, 03:30 MDT → July 5). Track: **Google DeepMind — Statement Five (Edge / On-Device)**: *"Best mobile, web, or edge application running Gemma locally for offline, privacy-first inference."*

> **Scope statement:** Helius is a navigation, signaling, and procedural field-tools agent. It is deliberately **non-medical and non-diagnostic** — it refuses medical prompts by design and shows that refusal as a feature. Deterministic local tools are orchestrated by Gemma; the model never invents facts the tools can compute.
